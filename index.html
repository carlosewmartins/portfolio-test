_projetos

Cadastro Ninja

Cadastro Ninja é uma API REST para gerenciamento de ninjas, desenvolvida em Java com Spring Boot. Conta com integração a banco de dados e uma interface web construída com Thymeleaf, permitindo cadastro, listagem e gerenciamento dos personagens.

MovieFlix

MovieFlix é um protótipo de sistema de streaming desenvolvido em Java puro, com interface gráfica construída em Swing. Permite navegação por catálogo de filmes, simulação de cadastro e reprodução, servindo como estudo de POO e interfaces desktop.

Heropedia

Heropedia é uma enciclopédia interativa de heróis e vilões. Criada em HTML, CSS e JavaScript, traz páginas temáticas com design responsivo e animações de entrada, oferecendo uma experiência leve e dinâmica para explorar personagens.

Batalha Naval

Batalha Naval em Python (console) com tabuleiro 10×10. O jogador posiciona 5 navios, a máquina faz o mesmo de forma oculta e os turnos se alternam com validação de entrada, feedback visual (X = acerto, O = erro) e condição de vitória automática. Ideal para praticar lógica, matrizes e interação no terminal.
